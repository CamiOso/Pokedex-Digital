 0 Número de Pokémon a omitir al inicio

Throws:
Si falla la conexión o la respuesta no es válida
Type
Error
Returns:
- Promesa que resuelve en un objeto con: - `count`: número total de Pokémon - `next`: URL del siguiente grupo (puede ser null) - `previous`: URL del grupo anterior (puede ser null) - `results`: Array de objetos `{ name, url }`
Type
Promise.<Object>
Examples
